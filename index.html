About

WebPonize is a Mac OS App for converting images into WebP.

Download

Download webponize.tar.gz

Installation

  1. Extract downloaded webponize.tar.gz.
  2. Move WebPonize.app into Application folder (`/Applications`).

Usage

Just drag and drop your images into WebPonize. It accepts PNG, JPEG, animated (or not) GIF.

You can confirm how file size is reduced.
